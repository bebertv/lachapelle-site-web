# 🤝 Guide de Contribution

Merci de votre intérêt pour le site de l'église Saint-Pierre de Lachapelle !

## 🎯 Comment Contribuer

### 1. Corrections et Améliorations
- Corrections typographiques
- Améliorations du responsive design
- Optimisations des performances

### 2. Traductions
Le site supporte 5 langues :
- Français (FR) ✓ Complet
- English (EN) ⏳ À traduire  
- Nederlands (NL) ⏳ À traduire
- Deutsch (DE) ⏳ À traduire
- 中文 (ZH) ⏳ À traduire

### 3. Nouvelles Fonctionnalités
- Formulaire de contact
- Système de réservation
- Gallery photo avancée
- Blog actualités

## 📋 Processus de Contribution

### Pour les petites modifications :
1. Fork le projet
2. Crée une branche : `git checkout -b feature/ma-contribution`
3. Commit : `git commit -m "Ajoute ma contribution"`
4. Push : `git push origin feature/ma-contribution`
5. Crée une Pull Request

### Pour les traductions :
1. Utilise le sélecteur de langue existant
2. Ajoute les fichiers de traduction dans `/translations/`
3. Test sur plusieurs devices

## 🧪 Testing

Avant de soumettre :
- [ ] Testé sur Chrome/Firefox/Safari
- [ ] Test responsive (mobile/tablette/desktop)
- [ ] Validation HTML/CSS ([W3C Validator](https://validator.w3.org/))
- [ ] Performance (Lighthouse)

## 🎨 Guidelines Design

### Couleurs (Respecter la charte)
```css
--color-primary: #8b6914    /* Or baroque */
--color-accent: #c9a227     /* Doré */
--color-bg: #faf8f5         /* Ivoire */
```

### Typographie
- Titres : Cormorant Garamond
- Corps : Outfit
- Taille minimum : 16px pour l'accessibilité

## 📝 Commit Messages

Utilisez le format conventionnel :
```
feat: ajoute la fonctionnalité X
fix: corrige le bug Y
docs: met à jour la documentation
style: changement cosmétique
```

## 🔍 Code Review

Toutes les contributions passent par une review pour :
- Qualité du code
- Respect du design
- Accessibilité
- Performance
